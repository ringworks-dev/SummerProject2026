# Summer Project 2026

> Build. Learn. Improve.

## 概要

Summer Project 2026は、2026年夏に実施する個人学習・開発プロジェクトです。

AI支援ツールを活用しながら、アプリケーションの設計、実装、改善、記録までを一連の開発プロセスとして経験することを目的としています。

このリポジトリでは、完成した成果物だけでなく、試行錯誤の過程、技術選定、失敗、改善内容も記録します。

## 目的

- AIを活用した開発フローを実践する
- ReactとTypeScriptを使ったアプリ開発を経験する
- GitとGitHubを継続的に使う
- 設計、実装、検証、改善の流れを身につける
- 言語や技術による実装方法の違いを比較する
- 学習内容をRingWorksの記事として整理する

## メインプロジェクト

### Headache Logger

頭痛や片頭痛の状態を記録し、振り返るためのログアプリを制作します。

予定している主な機能：

- 頭痛の発生日時を記録
- 痛みの強さや症状を記録
- 服薬内容を記録
- 記録の一覧表示
- 記録の編集・削除
- データの保存
- 記録内容の集計・可視化

## 使用予定の技術

- React
- TypeScript
- HTML
- CSS
- Git
- GitHub
- Claude Code
- Visual Studio Code

必要に応じて、PythonやGoなどを使った小規模な比較実装も行います。

## 進め方

1. プロジェクトの目的と要件を整理する
2. Headache LoggerのMVPを設計する
3. ReactとTypeScriptで実装する
4. 動作確認と改善を行う
5. 別言語や別構成での実装を試す
6. 開発過程と学習内容を記事にまとめる

## Overview

Summer Project 2026 is a personal software development and learning project.

The purpose of this project is to practice application design, implementation, testing, improvement, and documentation with AI-assisted development tools.
