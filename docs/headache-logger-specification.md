# Headache Logger Specification

## 1. Overview（概要）

Headache Loggerは、毎日の予防薬の服用状況やメモを記録し、
頭痛が発生した場合には痛みの記録を追加できるスマートフォン向けログアプリです。

## 2. Purpose（目的）

### User Perspective

自分にとって使いやすい頭痛ログアプリを開発し、日々の症状や服薬状況を継続して記録・振り返ることで、自分に合った記録方法を検証することを目的とします。

### Learning Perspective

React・TypeScriptを用いたアプリケーション開発を通して、
設計から実装、改善までの開発プロセスを学ぶ。

## 3. Target User（対象）

Headache Loggerは、頭痛の記録を「頑張って続ける」のではなく、
「気付いたときに気軽に残せる」ようなアプリを求めている人を対象としています。

複雑な入力ではなく、必要な情報を素早く記録し、
あとから振り返りやすいことを重視しています。

## 4. Functional Requirements（機能要件）

### 4.1 Daily Log

- ユーザーは日付ごとにDaily Logを記録できる
- Daily Logは、服薬状況、メモ、Pain Logのいずれかが初めて記録された時点で作成される
- Daily Logは1日につき1件作成される。
- Daily Logは、頭痛の有無にかかわらず毎日使用できる
- ユーザーは予防薬の服用状況を朝・昼・夜ごとに記録できる
- ユーザーは服用したタイミングで、それぞれの項目を個別にチェックできる
- 朝・昼・夜の服用記録は、後から変更できる
- ユーザーはその日のメモを記録できる
- ユーザーはDaily Logに痛みログを追加できる
- ユーザーは誤って記録した服薬チェックを解除できる
- 記録を削除する前に確認を表示する

### 4.2 Pain Log

- ユーザーは痛みが発生した時点で、その日のDaily LogにPain Logを追加できる
- 1つのDaily Logに複数のPain Logを追加できる
- Pain Logの開始時刻には、現在時刻を初期値として設定する
- ユーザーは開始時刻を修正できる
- ユーザーは痛みの強度を5段階で記録できる
- ユーザーは終了時刻を後から追加できる
- 終了時刻が未入力の場合、痛みが継続中であることを表示する
- 終了時刻には、開始時刻より前の時刻を設定できない
- ユーザーはPain Logに頓用薬の服用記録を追加できる
- ユーザーはPain Logにメモを記録できる
- ユーザーはPain Logを編集・削除できる
- ユーザーはPain Log作成後にも頓用薬の記録を追加できる
- ユーザーは頓用薬の記録を編集・削除できる
- 記録を削除する前に確認を表示する

### 4.3 Medication

- ユーザーは予防薬の服用状況をDaily Logに記録できる
- ユーザーはPain Logごとに頓用薬を追加できる
- 頓用薬は処方薬または市販薬として記録できる
- ユーザーは頓用薬の名称と服用時刻を記録できる
- 1つのPain Logに複数の頓用薬を追加できる

### 4.4 Calendar

- ユーザーは月単位のカレンダーを表示できる
- カレンダーを開いた時点で本日を強調表示する
- Daily Logが存在する日にはマークを表示する
- Pain Logが存在する日には頭痛の記録があることを表示する
- ユーザーは日付を選択して、その日の記録を表示できる

### 4.5 Record Management

- ユーザーは過去の記録を閲覧できる
- ユーザーは過去の記録を編集できる
- ユーザーは不要な記録を削除できる

## 5. Screen List（画面一覧）

### 5.1 今日

アプリ起動時に最初に表示する画面。
このアプリのホーム画面とする。

- 本日の日付
- 予防薬の朝・昼・夜チェック
- その日のメモ
- Pain Log一覧
- Pain Log追加操作
- 継続中のPain Logに対する終了記録操作

### 5.2 カレンダー

- 月単位のカレンダー
- 本日の強調表示
- Daily Logが存在する日の表示
- Pain Logが存在する日の表示
- 日付選択による記録表示

### 5.3 日付別記録

- 選択した日付のDaily Log
- 服薬状況
- メモ
- Pain Log一覧
- 過去記録の編集

### 5.4 Pain Log入力・編集

- 開始時刻の編集
- 終了時刻の追加・編集
- 痛みの強度の編集
- 頓用薬の追加・編集・削除
- メモの編集
- Pain Logの削除
